# arxiv_scraper
A simple package to scrape arxiv papers, use openai to summarize them, and put them in a google sheet.

Requires a `config.env` file, a postgres db setup, and a google sheets api key.:
```
OPENAI_API_KEY=
DB_NAME=
DB_USER=
DB_PASSWORD=
SHEETS_CREDENTIALS_FILE=
SHEETS_NAME=
SHEETS_WORKSHEET=
```
